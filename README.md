### Практическое задание #9 - Streamlit веб-приложение

Веб-приложение для предсказания стоимости квартир в Москве на основе обученной модели машинного обучения.

**Содержимое папки:**
- `main.py` - Streamlit приложение с интерфейсом для предсказания цены
- `Model_training.ipynb` - Ноутбук для обучения модели Random Forest
- `rf_model.pkl` - Предобученная модель Random Forest
- `realty_data.csv` - Датасет с данными о недвижимости
- `requirements.txt` - Зависимости проекта

**Функционал:**
- Интерактивный интерфейс с ползунками для выбора параметров
- Предсказание стоимости квартиры на основе:
  - Количества комнат (1-20)
  - Номера этажа (1-100)
  - Площади объекта (15-5000 м²)

**Запуск:**
```bash
cd "Практическое задание_9"
pip install -r requirements.txt
streamlit run main.py
```

**Для обучения модели:**
- Необходимо скачать [датасет](https://disk.yandex.ru/d/XDNnTb8JdvYpCA) для запуска `Model_training.ipynb`
- Или использовать предобученную модель `rf_model.pkl` (можно скачать [здесь](https://niuitmo-my.sharepoint.com/:u:/g/personal/askernagoev_niuitmo_ru/IQAUA1jAY9rVSYaVVfOKKsorAbA0bxxRHIEgkl23pDGR6vU?e=4wH6Hs))

### Практическое задание #10 - FastAPI REST API

REST API для предсказания стоимости квартир в Москве с использованием FastAPI.

**Содержимое папки:**
- `main.py` - FastAPI приложение с REST API endpoints
- `rf_model.pkl` - Предобученная модель Random Forest
- `requirements.txt` - Зависимости проекта
- `Метод get.png`, `Метод post.png`, `Начальная страница.png` - Скриншоты работы API

**API Endpoints:**
- `GET /health` - Проверка работоспособности сервера
- `GET /predict_get` - Получение предсказания с фиксированными параметрами
- `POST /predict_post` - Получение предсказания с пользовательскими параметрами
  - Параметры запроса:
    - `total_square` (float) - площадь объекта
    - `rooms` (int) - количество комнат
    - `floor` (int) - номер этажа

**Запуск:**
```bash
cd "Практическое задание_10"
pip install -r requirements.txt
python main.py
```

API будет доступен по адресу: `http://localhost:8000`

**Документация API:**
После запуска сервера доступна автоматическая документация:
- Swagger UI: `http://localhost:8000/docs`
- ReDoc: `http://localhost:8000/redoc`

**Пример использования POST запроса:**
```bash
curl -X POST "http://localhost:8000/predict_post" \
  -H "Content-Type: application/json" \
  -d '{"total_square": 75.0, "rooms": 2, "floor": 5}'
```
